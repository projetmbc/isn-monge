Une petite démonstration de ce que permet CSS3

Les codes CSS utilisés dans cette page vont bien au-delà de ce qui est attendu en ISN au lycée.

Ceci étant dit, je vous propose cet exemple qui fait suite à une question d'un collègue désirant cacher par défaut les démonstrations dans ses documents HTML, tout en ayant la possibilité de les faire apparaître via un simple clic.

Les liens indiqués dans le code CSS de cette page, et un peu de réflexion ont permis d'aboutir à l'effet suivant (demandez-moi si vraiment vous voulez comprendre le code CSS utilisé et que vous ne comprenez pas de vous-même).

Cliquer sur « Irrationalité de π » ci-dessous et vous saurez pourquoi le nombre π est irrationnel.

Mettre un CHEVAL sur un OISEAU étant irrationnel, nous pouvons affirmer que CHEVAL / OISEAU est irrationnel.

Ensuite CHEVAL = C × H × E × V × A × L = V × A × C × H × E × L = VACHE × L où l'on peut constater que VACHE = BÉTA × π car une vache est une bête à pis, c'est à dire 1 BÉTA π. Notons que de même OISEAU = BÉTA × L.

Les deux points précédents nous donnent l'irrationnalité de CHEVAL / OISEAU = (BÉTA × π × L) / (BÉTA × L) = π.
